這是一個 Blog 網站

目前具有以下功能
1. 登錄/注冊功能
2. 上傳/編輯/刪除文章
3. 首頁依照人氣 / 文章類別 分類文章
4. 點讚
5. 文章搜尋
6. 藉AWS S3平台實現 部份用戶資料介面功能(更換頭像)
7. 透過 JWT 實現安全遊覽網頁的會員區域
8. 留言區
9. 個人檔案(更換頭像/更換密碼/顯示或修改個人資料/展示用戶數據)

正在完善的功能
1. 文本編輯器
2. 收藏功能
3. 通知功能

使用語言
- Frontend: TypeScript
- Backend: Node.js
- Database: MongoDB
- Storage: AWS S3
